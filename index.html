Key Projects

Python V-SLAM Algorithm (Ongoing Thesis)

My thesis, 'Leveraging Geometric Priors for Robust Visual Odometry in 3D Reconstruction', involves a deep dive into Visual SLAM. I'm implementing state-of-the-art algorithms for keypoint matching, predicting 6D camera pose using differential algorithms over RANSAC/USAC, and using Delaunay triangulation to project 2D points to 3D, minimized by Reprojection error. Absolute pose is found via the PnP algorithm and verified with ATE calculations.

Python Structure from Motion (SfM) Algorithm

Implemented the SfM pipeline as described in the "SFM revisited" paper using standard COLMAP datasets. While the paper uses a Multi-View Stereo (MVS) algorithm for matching, my implementation uses a Deep-learning-based View Stereo (DVS) approach, offering a modern take on the classic pipeline.

Dynamic Mobile Robot Navigation in ROS

This project involved simulating a mobile robot navigating a 2D map with both static and dynamic obstacles. I utilized ROS for control, Gazebo for physics simulation, and advanced deep learning techniques like semantic segmentation and keypoint estimation for environment perception. The research was published at IEEE CONECCT 2024.

Road Width Estimation using GEE & Deep Learning

Collected and manually annotated geospatial data from Google Earth Engine (GEE) along a national highway. I trained a deep residual neural network to perform precise road segmentation from this data, enabling accurate estimation of road width—a key task in infrastructure monitoring and analysis.
